CQT 与小波：匹配人耳的频谱分析

为什么 FFT 的等带宽 bin 与人耳错配，而恒定 Q 变换（CQT）与小波如何解决这个问题。

1. 核心矛盾：等带宽 bin vs. 对数感知

人耳的频率感知是对数的

100 → 200 Hz 和 1000 → 2000 Hz 听起来是同样的距离（都是八度）。

临界带宽（ERB）也随频率增长而变宽：

  • 100 Hz 处 ERB ≈ 35 Hz（很窄）
  • 1 kHz 处 ERB ≈ 132 Hz
  • 5 kHz 处 ERB ≈ 668 Hz（很宽）

耳朵天然是一个恒定 Q 分析仪——$Q = f / \Delta f \approx \text{常数}$。

标准 FFT 使用等间距 Hz bin

1024 点 FFT @ 44.1 kHz → 每个 bin 固定 43 Hz 宽。

  • 100 Hz 处：43 Hz bin > 35 Hz ERB 太粗
  • 1 kHz 处：43 Hz bin < 132 Hz ERB 尚可
  • 5 kHz 处：43 Hz bin ≪ 668 Hz ERB 过细（浪费）

FFT 在低音区分辨率不足，在高音区维度冗余——与人耳完全错配。

3. 时频分辨率的本质：Heisenberg-Gabor 不确定性

每个分析"原子"在时频平面上占据一个确定的最小面积：

$$\Delta t \cdot \Delta f \geq \frac{1}{4\pi}$$

不同分析方法在时频平面上画出的Heisenberg 盒形状不同：

FFT / STFT
固定矩形盒——所有频率处时间、频率分辨率相同。低音区Δt太短（Δf太粗），高音区Δf太细（Δt太长）。
CQT
盒宽与频率成正比——低音区长Δt(高Δf精度)，高音区短Δt(高时间精度)。恰好匹配听觉。
连续小波
盒连续变化——每个尺度有最优Δt-Δf平衡。Gabor 小波达到等号(=1/4π)。

5. CQT 的数学原理

核心思想：让 bin 间距随频率等比增长

CQT 的第 k 个频率 bin 中心为：

$$f_k = f_{\min} \cdot 2^{k / B}$$

其中 $B$ = 每八度的 bin 数（如 $12$ = 每半音一个独立 bin，$24$ = 每 quarter-tone）。

CQT 的关键特性——恒定 Q 值：

$$Q = \frac{f_k}{\Delta f_k} = \text{常数}$$

这意味着在高频端 $\Delta f$ 大（时间分辨率高，适合瞬态），低频端 $\Delta f$ 小（频率分辨率高，适合精确音高）——恰好匹配人耳的临界带宽分布。

与小波的关系

连续小波变换（CWT）是 CQT 的连续极限：

$$W(a, b) = \int_{-\infty}^{\infty} s(t) \cdot \frac{1}{\sqrt{a}} \; \psi\!\left(\frac{t - b}{a}\right) dt$$

$a$ = 尺度（↔ 频率的倒数），$b$ = 平移（↔ 时间位置）。所有分析函数由单一母小波缩放而来——本质上是自相似的、跨频率的恒定 Q 分析。

7. 实质：窗口形状的自适应

低频 = 长窗 = 高 Δf 精度

100 Hz 的正弦波需要 ~50 ms 才能"看准"频率。CQT/小波在低频自动放宽时间窗——你能精确判断音高，正如听觉系统在低频端的高频率辨别能力。

高频 = 短窗 = 高 Δt 精度

5 kHz 的瞬态脉冲只需 ~2 ms 的窗。CQT/小波在高频自动收紧时间窗——你能精确捕获起奏和瞬态，正如听觉系统对打击乐的快速响应。
