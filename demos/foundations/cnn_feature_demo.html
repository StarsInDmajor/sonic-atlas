从物理声谱到感知音色——卷积神经网络如何自动发现人类音色空间的结构。基于 Pons et al. (2017)、Kong (2011)、SpectroFusionNet (2025) 等研究。

5. 平移等变性：CNN 为何天然适合音色

频率平移 = 同一乐器演奏不同音高

CNN 的平移等变性在音频中具有清晰的物理含义。交互演示：拖动"音高"滑块，观察谐波列在 Mel 声谱图上的垂直平移——CNN 的卷积核在任何位置都能检测到相同的模式。

频率轴平移 → 音高不变
同一乐器演奏 A3 (220 Hz) 或 C4 (262 Hz)，谐波列在 Mel 轴上整体上移/下移。CNN 的卷积核在任何垂直位置都能检测到相同的谐波模式——自然习得音高不变的音色识别。
时间轴平移 → 起奏对齐不变
同一个钢琴起奏模式出现在声谱图的第 100 帧或第 500 帧，CNN 的时间轴卷积核都能检测到——自然习得时间对齐不变性。无需 DTW 或显式对齐。

8. 2D-CNN（声谱图）vs 1D-CNN（波形）

2D-CNN + 声谱图 主流

  • 输入：log-Mel 声谱图 $(M \times T)$
  • 复用图像分类的全部成果（VGG、ResNet 迁移学习）
  • 小样本有效——迁移学习 + 预训练权重
  • 工业标准：98%+ 乐器分类（受控条件）
  • 自动习得频率-时间平移等变性

1D-CNN + 原始波形 前沿

  • 输入：44.1 kHz 样本序列
  • 第一层学习类滤波器的冲激响应
  • 理论上有能力学习任意基底（不限 Fourier）
  • 需要数量级更大的数据才能泛化
  • 主要用于语音合成等生成式任务（WaveNet）
当前工业实践：声谱图 → 2D CNN 是绝对主流方案。1D-CNN 在判别式任务上远不如声谱图 + 预训练 CNN 的迁移学习，但对生成式任务（如 WaveNet 语音合成、音乐生成）不可或缺。

总结

频率-强度-时间框架 → 声谱图 → 2D CNN 构成了当前音频音色理解的最优计算路径。 CNN 在此路径上自组织地学会检测频谱包络（对应亮度）、时间包络（对应起奏形态）、和频谱纹理（对应规则度）——这三者恰好覆盖了 Grey (1977) 以来 MDS 音色空间的核心维度。物理框架提供了完备的输入基底，CNN 提供了从物理维度到感知维度的自动映射——这一组合在最基本的层面上是"正确的"，且在实践中已被证明。

核心公式： 声谱图像素 $(f, t)$ 对应物理三元组 $(f, t, a)$。CNN 的 2D 卷积核 $\mathbf{K} \in \mathbb{R}^{k_f \times k_t}$ 在 $(f, t)$ 近邻上计算

$$y_{i,j} = \sigma\!\Bigl(\sum_{m=0}^{k_f-1}\sum_{n=0}^{k_t-1} K_{m,n} \cdot x_{i+m,\, j+n} + b\Bigr)$$

其中 $x_{i,j}$ 是 Mel 声谱图在频带 $i$、时间帧 $j$ 处的对数能量，$\sigma$ 是非线性激活函数。这个操作的物理意义：在频率-时间近邻中检测局部模式——等价于听觉皮层对频谱-时间纹理的响应。
